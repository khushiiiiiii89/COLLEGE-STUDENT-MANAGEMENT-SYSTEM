**# COLLEGE-STUDENT-MANAGEMENT-SYSTEM**
The College Management System is a web-based application designed to simplify and digitize the daily administrative and academic operations of a college. The system provides a centralized platform to manage student records, user roles, academic data, fees, and performance details in a secure and efficient manner.

**# College Student Management System**

A MERN Stack web application to manage students, marks, fees, and roles.

**## Features**
- JWT Authentication
- Admin & Teacher roles
- Student Management
- Marks & Fee Tracking

**## Tech Stack**
MongoDB, Express, React, Node.js

**## Run Project**
Backend:
npm install
npm run start

Frontend:
npm install
npm start
